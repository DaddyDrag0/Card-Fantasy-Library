# Card images

Put card images in this folder using the card id as the filename.

Examples:

- `hobbit.png`
- `squire.png`
- `goblin.png`
- `orc-warrior.png`

The site checks these extensions automatically:

- `.png`
- `.webp`
- `.jpg`
- `.jpeg`

You can also add an `image` or `imagePath` field to a card in the JSON if a card needs a custom image path.
